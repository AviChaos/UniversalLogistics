## Overview

Universal Logistics is a Satisfactory mod that expands the game's logistics system while preserving the original gameplay philosophy.

Rather than replacing Coffee Stain Studios' logistics mechanics, Universal Logistics extends them with additional building orientations and quality-of-life improvements that feel like natural additions to the base game.

Every feature is guided by one principle:

> **"Expand the possibilities. Preserve the experience."**

# Planned Features

## Logistics Expansion

- Multiple Splitter Orientations
- Multiple Merger Orientations
- Horizontal Logistics
- Vertical Logistics

All variants preserve vanilla functionality.

Splitters remain:

- 1 Input
- 3 Outputs

Mergers remain:

- 3 Inputs
- 1 Output

Only the physical orientation changes.

# Design Philosophy

Universal Logistics is built around five principles.

- Preserve vanilla gameplay
- Expand creativity instead of power
- Reuse existing game systems whenever practical
- Maintain progression balance
- Prioritize quality over quantity
